# Group-Chatroom
server based on Linux TCP socket; client based on QT QTcpSocket; database MySQL

## Supporting Functions
* Show Old messages
* Send messages file(*.all)
* Show online user
* Send  to offline user
* Server with log
* Set ui config (color)
* Set number of old messages
* Change password
